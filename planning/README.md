# Planning Folder Index

## Purpose
Contains the approved strategy Claude should use when building.

## Files
- `brand-positioning.md` — target audience and differentiation
- `site-architecture.md` — page sections and purpose
- `offers-and-packages.md` — service/package framing
- `copy-deck.md` — approved homepage copy
- `asset-inventory.md` — approved images, videos, logos, and missing assets

## Rule
When building the site, prefer these files over raw research notes.
