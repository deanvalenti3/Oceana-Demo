# Execution Plan — Oceana Homepage MVP

## Goal
Build a high-converting one-page website demo based on approved strategy.

## Core Conversion Flow
Hero → Offer → Crew → Testimonials → Inquiry Form

## Build Priority Order

### 1. Hero Section
- Background image: yacht + glacier
- Headline: "Your Own Private Alaska — No Crowds, No Compromises"
- CTA: Check Availability — 2026 Season

### 2. Offer Section
- 8 Days / 7 Nights private expedition
- Clear inclusions
- From-price framing
- CTA repeat

### 3. Crew Section (Trust Anchor)
- 4 crew members with names, roles, credentials
- Chef highlighted

### 4. Testimonials
- Pull EXACT quotes from source
- No rewriting

### 5. Inquiry Form
- Name
- Email
- Preferred dates
- Simple submit

### 6. Sticky CTA Bar
- Persistent: "Check Availability — 2026"

## Rules
- No design fluff
- No unnecessary sections
- Everything must support conversion

## Assets Needed
- Hero image (highest priority)
- Crew photos
- Testimonial copy
